### CineSearch – Movie Explorer
![CineSearch Screenshot](https://via.placeholder.com/600x300.png?text=CineSearch+Movie+App)

**Tech:** HTML, CSS, JavaScript, OMDB API

**Description:**  
CineSearch is a simple and responsive web application that allows users to search for movies and view their details and posters in real-time. It fetches data from the OMDB API and dynamically displays search results.

**Features:**  
- Search movies by title  
- Display poster, release year, and movie details  
- Responsive design for mobile and desktop  
- Vanilla JavaScript DOM manipulation
