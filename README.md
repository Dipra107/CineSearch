**Description:**  
CineSearch is a responsive movie search web app where users can search for movies and view posters and details fetched live from OMDB API.  

**Features:**  
- Search movies by title  
- View poster, release year, and movie details  
- Responsive design for desktop and mobile  
- Vanilla JavaScript DOM manipulation
